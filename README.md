# To-Do List in Python
 
This project is a simple To-Do List application developed in Python. It allows users to add, view, and delete tasks through a Command Line Interface (CLI).
 
## Features
- Add tasks.
- View tasks.
- Delete tasks.
- Exit the application.
 
## Requirements
- Python 3.x
